HR Analytics Dashboard

Project Overview

The HR Analytics Dashboard is an interactive Power BI dashboard developed to analyze employee data and provide insights into workforce trends, employee attrition, and job satisfaction. The dashboard helps organizations understand employee behavior and make data driven decisions.

Objective

To analyze employee information and identify key factors affecting attrition, employee demographics, and job satisfaction across different departments and age groups.

Dashboard Features

Key Performance Indicators

- Overall Employees
- Attrition Count
- Attrition Rate
- Active Employees
- Average Employee Age

Analysis Included

- Department Wise Attrition
- Employee Distribution by Age Group
- Job Satisfaction Rating by Job Role
- Education Field Wise Attrition
- Attrition Rate by Age Groups
- Gender Based Analysis

Tools and Technologies Used

- Power BI
- Microsoft Excel
- Data Cleaning and Transformation
- Data Visualization
- DAX Functions
- Power Query

Insights Generated

- Identified departments with the highest employee attrition.
- Analyzed employee distribution across different age groups.
- Measured job satisfaction ratings for various job roles.
- Evaluated the impact of educational background on attrition.
- Compared attrition rates among different age categories.

Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Modeling
- Dashboard Design
- Data Visualization
- DAX Calculations
- Business Intelligence Reporting

Project Outcome

This dashboard provides a clear understanding of workforce trends and employee attrition patterns, enabling management to take strategic decisions for improving employee retention and organizational performance.

Suitable For

This project is designed as a portfolio project for Data Analyst and Business Intelligence Fresher roles and demonstrates practical experience in Power BI dashboard development and HR data analytics.
